HEVC Transcode Killer

Prevent HEVC files from transcoding audio.
Prevent HEVC files from transcoding video.
Prevent paused videos from sitting idle for too long.
Will popup audio transcode message (Not HEVC Only)
Will popup video transcode message (Not HEVC Only)
Nag user of transcode every N minute(s)

Exclude folders:

Message to be shown when both audio and video is disabled.
Message to be shown when both audio and video nag is enabled.
Message to be shown when only video is disabled.
Message to be shown when only video nag is enabled.
Message to be shown when only audio is disabled.
Message to be shown when only audio nag is enabled.
Message to be shown when video is paused too long.
